# Design Guardrails

## Primary Principle

Usability first, novelty second.

## Preserve By Default

- Current login visual identity
- Core timeline information density
- Reservation block readability
- Mobile friendliness

## Preferred Improvements

- clearer hierarchy
- better spacing rhythm
- stronger section grouping
- more consistent interaction states
- cleaner modal layouts

## Avoid

- unnecessary redesign of stable screens
- decorative complexity that harms scanning
- visual changes that imply behavior changes the code does not support
