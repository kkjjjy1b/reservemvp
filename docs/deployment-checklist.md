# Deployment Checklist

배포 실행 직전에 빠르게 확인하기 위한 체크리스트다.

## 준비
- `project-root` Git 원격 저장소 URL 확보
- 기본 배포 브랜치 결정 (`main` 권장)
- 운영 `DATABASE_URL` 확보
- 운영 `SESSION_SECRET` 확보
- 프로필 이미지 업로드를 사용할 경우 운영 `BLOB_READ_WRITE_TOKEN` 확보
- 예약 참여자 메일 발송을 사용할 경우 운영 `RESEND_API_KEY` 확보
- 예약 참여자 메일 발송을 사용할 경우 운영 `RESEND_FROM_EMAIL` 확보
- 배포 대상 플랫폼 결정
- 운영 도메인 결정

## DB
- 운영 PostgreSQL 생성
- migration 적용
- 회의실 데이터 입력
- 사용자 데이터 입력

## 앱
- `project-root`가 독립 Git 저장소로 관리되는지 확인
- 원격 `origin` 연결 확인
- `npm run build` 통과
- 최신 코드 머지 완료
- 배포 대상 브랜치 확정

## 기능 검수
- 로그인
- 자동 로그인
- 타임라인 조회
- 예약 생성
- 예약 수정
- 예약 취소
- 내 정보 수정
- 프로필 이미지 업로드
- 참여자 검색
- 참여자 포함 예약 생성 / 수정
- 참여자 예약 알림 메일 발송
- 내 예약 목록 조회

## 공개 직전
- 테스트 계정 제거 또는 운영 계정으로 교체 여부 확인
- 운영용 초기 비밀번호 정책 확인
- 로그 확인 경로 확인
- 장애 시 담당자 확인
